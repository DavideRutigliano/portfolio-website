## Technical Details

The MCP (Model Context Protocol) server acts as a bridge between observability data and AI models, exposing structured tools for:

- Topology exploration and component search
- Metric analysis and anomaly detection  
- Monitor status and health state queries
- Trace analysis for distributed systems

Built with **Go**, integrating with **SUSE Observability (StackState)** for federated multi-cluster observability.
